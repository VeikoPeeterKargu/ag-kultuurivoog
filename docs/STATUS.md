# Project Status

## Arhitektuur
- **Backend:** FastAPI
- **Frontend:** Tailwind CSS (Vanilla/Static)
- **Database:** PostgreSQL (Hosted on Render)
- **Scraper:** Python scripts (Migrated to GitHub Actions)

## Mis on valmis
- **API:** Toimiv FastAPI rakendus andmete serveerimiseks.
- **UI:** Tailwindiga kujundatud kasutajaliides sündmuste vaatamiseks.
- **Andmebaas:** Skeem ja vaated (views) on paigas ja töötavad.
- **Funktsionaalsus:** ICS kalendri eksport töötab.

## Blokeering
- **Teater.ee 403 viga:** Pilveserverite (Render, potentsiaalselt ka GitHub Actions) IP-aadressid on blokeeritud. See takistab andmete automaatset uuendamist pilves.
